# CommunityConnect

**CommunityConnect** is an AI-powered civic engagement web application designed to improve communication between residents and municipalities by providing a simple, transparent, and efficient platform for reporting service delivery issues. The platform is specifically designed for rural and underserved communities, such as **Mvutshini, Port Shepstone, KwaZulu-Natal**, where residents often experience challenges including water shortages, electricity outages, damaged roads, illegal dumping, poor waste management, and limited communication from local authorities.

CommunityConnect leverages Artificial Intelligence to improve the quality of citizen reports, provide real-time updates, and help municipalities prioritize infrastructure maintenance while promoting transparency and accountability.

---

# Project Overview

Many South African communities face ongoing service delivery challenges that are difficult to report and track effectively. Residents often have no simple way of knowing whether their complaints have been received or when they will be resolved.

CommunityConnect addresses this problem by providing an AI-powered digital platform that allows residents to:

- Report municipal issues quickly.
- Track the progress of submitted reports.
- Receive community alerts.
- View reports on an interactive map.
- Access ward councillor and municipal contact information.
- Participate in community discussions and petitions.
- Receive AI-powered assistance when submitting reports.

The application aims to improve communication between communities and local government while empowering citizens to actively participate in improving their neighbourhoods.

# Setup Instructions

There are two ways to use **CommunityConnect**.

## Option 1: Run the Project Locally Using Visual Studio Code (Recommended for Developers)

### Step 1: Clone the Repository

Open **Visual Studio Code**, launch a new terminal, and run:

```bash
git clone https://github.com/charmaine-dlamini-18/AI-Powered-Assistant.git
```

### Step 2: Navigate to the Project Folder

```bash
cd AI-Powered-Assistant
```

### Step 3: Open the Project in Visual Studio Code

If the project is not already open in VS Code, open the cloned folder.

You can also run:

```bash
code .
```

### Step 4: Install Dependencies

Install all required project dependencies by running:

```bash
npm install
```

### Step 5: Configure Environment Variables (If Required)

Create a `.env` file in the project root directory and add the required API keys.

Example:

```env
OPENAI_API_KEY=your_openai_api_key
GOOGLE_MAPS_API_KEY=your_google_maps_api_key
FIREBASE_API_KEY=your_firebase_api_key
```

### Step 6: Start the Development Server

Run either of the following commands:

```bash
npm run dev
```

or

```bash
npm start
```

### Step 7: Open the Application

Once the development server starts successfully, open your browser and navigate to the URL displayed in the terminal, typically:

```
http://localhost:5173
```

or

```
http://localhost:3000
```

depending on your development environment.

# Responsible AI

CommunityConnect uses Artificial Intelligence to assist users in reporting municipal issues and generating summaries. AI-generated suggestions, translations, classifications, and recommendations are intended to support—not replace—human judgment. Users and municipal officials should verify AI-generated content before making decisions or taking action.
